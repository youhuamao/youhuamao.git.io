# 安装 Hexo 主题

  • 快速安装

打开主题目录，内有 example 文件夹，提供了配置文件的 demo 供参考。我写的时候跟个傻蛋似的不知道看

  • 安装依赖插件
    安装前，记得务必卸载掉默认的 hexo-renderer-marked ，以及别的 markdown 文件渲染器。

安装

用同上方法安装 hexo-autoprefixer，hexo-algoliasearch，hexo-symbols-count-time，hexo-feed

# 配置文件

  • 加入 markdown 配置，用来渲染 md 文件
  • 加入 minify 配置，压缩 css/js/html
  • autoprefixer 配置建议
  • algolia 配置建议
  • feed 配置建议

# 应用主题

  • 修改站点配置文件 <root>/_config.yml ，把主题改为 shoka
  • 修改主题配置
    主题配置的所有参数在 <root>/themes/shoka/_config.yml 文件中。
    为了方便主题升级，请在根目录新建一个 yml 文件，命名为 _config.shoka.yml
    也就是说，所有主题的自定义配置均保存于 <root>/_config.shoka.yml 文件。

# 将网站托管到 vercel

  • 打开 vercel，用 GitHub 登录，点击 Add new，再点击 project，找到你的博客仓库点击仓库旁边的 import，等待部署完成
  • 点击 Go to Dashboard，Settings，Domains，在 add 旁边新增一个域名，点击 add
  • 之后需要进行解析

# 通过自定义域名访问博客

  • 首先需要购买云服务器，我们以腾讯云服务器为例。
  • 接着在腾讯云购买一个属于你的域名
  • 在 vercel 中生成你的网址
  • 将网址在自己的域名下解析
