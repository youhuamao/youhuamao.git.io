  • 图像地址 src
  • 图像的替代文字 text (图片加载不出来会显示)

# 链接标签

# 文字超链接

  • <a href="path" target="目标窗口位置"> 链接文本或图像 </a>
  • href 链接路径
  • target 链接在哪个窗口打开 _blank: 另起窗口打开 _self: 当前窗口打开

# 图像超链接

  • <a href="path" target="目标窗口位置">
  • <img src="path" alt="text" title="text" width="x" height="y"/>
  • </a>

# 锚链接

  • 需要一个锚标记，使用 name 作为标记
  • <a name="top">dingbu</a>
  • 跳转到标记
  • <a herf="#top"> 回到 dingbu</a>
  • <a herf="连接标签.html#top"> 跳转到另外页面的锚点 </a>

  • 邮件链接: mailto:

# 列表标签

# 什么是列表

  • 列表就是信息资源的 - - 种展示形式。它可以使信息结构化和条理化，并以列表的样式显示出来，以便浏览者能更快捷地获得相应的信息

# 列表的分类

  • 有序列表 (1. 2. 3.)

  • <ol>
    <li>Java</1i>
    <li>Python</li>
    <li> 运维 </li>
    <li> 前端 </li>
    <li>C/c++</li>
    </ol>

  • 无序列表（. . .）

  • <ul>
    <li>Java</1i>
    <li>Python</li>
    <li> 运维 </li>
    <li> 前端 </li>
    <li>C/c++</li>
    </ul>

  • 定义列表

  • dl: 标签

  • dt : 列农名称

  • dd : 列表内容

  • <dl>
    <dt> 学科 </dt>
    <dd>Java</dd>
    <dd>Python</dd>
    <dd>L inux</dd>
    <dd>C</dd>
    <dt> 位置 </dt>
    <dd> 西安 </dd>
    <dd> 重庆 </dd>
    <dd> 新疆 </dd>
    </dl>
