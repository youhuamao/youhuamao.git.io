# 万能的 Map

  • 假设，我们的实体类，或者数据库中的表，字段或者参数过多，我们应当考虑使用 Map!
  • 使用 map 在写 sql 语句的时候，占位符内的名称可以与前边的不一致，类型一致即可
  • Map 传递参数，直接在 sq | 中取出 key 即可！[parameterType="map"]
    对象传递参数，直接在 sq | 中取对象的属性即可！[parameterType="Object"]
    只有一个基本类型参数的情况下，可以直接在 sq | 中取到！
    多个参数用 Map, 或者注解！

# 模糊查询
